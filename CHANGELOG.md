# Changelog — PromptForge

All notable changes to this project will be documented in this file.

## [0.1.0] — 2026-02-23

### Added

- Initial release: creative prompt engineering dashboard
- LLM-powered prompt refinement engine using Claude
- 8 platform adapters: Midjourney, Stable Diffusion, Runway, Kling, Adobe Firefly, Google Veo 3, Nano Banana, Grok Aurora (placeholder)
- 8 preset categories with 100+ presets: Aspect Ratio, Lighting, Camera + Lens, Film Stock, Atmosphere, Art Style, Composition, Color Palette
- Directed Q&A system for context gathering from vague prompts
- Dark-mode dashboard with amber forge theme
- Platform-specific prompt formatting
- Copy-to-clipboard output
- Zustand state management
- Next.js 15 App Router architecture
- Full 4 .md pattern (CLAUDE.md, cursor rules, skills, subagent)
- Workflow orchestration rules
- Task tracking infrastructure
